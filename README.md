# SvelteKit E-commerce

Full e-commerce storefront with product catalog, category filtering, Svelte 5 reactive cart store, and cart drawer checkout.

**Stack:** Sveltekit | **Complexity:** complex
**Tags:** `sveltekit`, `svelte`, `ecommerce`, `shop`, `cart`, `checkout`, `typescript`, `store`

## Quick Start

```bash
npm install
npm run dev
```

## Docker

```bash
docker compose up --build
```
